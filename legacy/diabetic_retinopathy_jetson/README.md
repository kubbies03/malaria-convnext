# Legacy Diabetic Retinopathy Jetson Package

This folder preserves an older Jetson Nano deployment workflow for diabetic retinopathy classification.

It is kept as historical project material only. The primary review path for this repository is the cleaned deployment package under `deploy/jetson_nano/`.

## Included Files

- `JETSON_DEPLOYMENT.md`
- `jetson_inference.py`
- `optimize_for_jetson.py`
- `requirements_jetson.txt`
- `install.sh.MD`

Some files in this folder have legacy formatting and encoding issues because they were preserved as-is instead of being rewritten into the main deployment package.
